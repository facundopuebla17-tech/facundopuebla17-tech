## 🚀 Proyectos Destacados

### 🤖 [Binance AI Trading Bot](https://github.com/facundopuebla17-tech/binance-ai-bot)
Sistema de trading algorítmico con motor de decisiones basado en Claude AI.
- Dashboard en tiempo real con 6 indicadores técnicos (RSI, EMA, MACD, Bollinger Bands, ATR)
- Integración con Binance API via HMAC-SHA256
- Backend Node.js/Express con autenticación, rate limiting y circuit breaker
- Análisis de sentimiento via Polymarket · Modos paper y live trading

**Stack:** `Python` `Node.js` `Claude API` `Chart.js` `Binance API` `Express`
